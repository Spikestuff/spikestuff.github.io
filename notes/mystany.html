The Endless Void That is Myst any%
(Are we already talking about this?)

So this nightmare title that's a 2 minute speedrun has a few differences.
You'd only know if you dedicated your time to it like Lizstar has... or myself.

So surely that means there are notable differences, and you'd be correct,
so we're going to do a "quick" rundown of this.


The Original Mac OS Release

So Myst was a title that was released on the MacOS in September 1993 and this is where the differences immediately start.
The MacOS version has an error, where in the final fireplace code there are two input sequences you can place in it, the correct one, and Page 148.

Enter the Mohawk Engine

Windows & Commodore Amiga

As of the release of the Windows 3.1 version a year later Cyan changed engines to Mohawk,
this release would basically be the ground works of many versions, and to knock out the ones that have no differences.

Well we're skipping the Windows 3.1 release, nothing changes except for the Page 148 oversight being removed.
Whilst here we're going to be skipping the Commodore Amiga version as well.


3DO, Sega Saturn and iPhone/Android

Meaning we're at the next release for the 3DO machine.
This version doesn't have zipping at all... so there's nothing else to talk about here.
Other "zipless" versions are the Sega Saturn and iPhone/Android releases.


Jaguar CD

Funnily enough the Jaguar CD version has a grand total of one zip in the any% speedrun,
and that's found between the Library and the Docks.

This is the only zip.
You have to walk everywhere else with this chonky controller.


Sony PlayStation

Now we've made it to the whiplash version.
PlayStation. "Zip Mode" is horrendously broken.

Once you enable that "Zip Mode", you're able to click almost anywhere and just be elsewhere.
You can go from the starting position to the Sunken Gear in one click,
compared to the actual traversal of five clicks.

You can practically instantly transmission anywhere and everywhere.


CD-i

And now we're at the CD-i version. We have two zips that no longer exist.

That's the movement from the Clock Tower to the crossroad towards the Generator and Log Cabin
and the movement from the Library to the Docks.


Nintendo DS

Nintendo DS is going to be mentioned here, it may be identical to the Windows release,
but besides the major difference of it being touch controls and not having a visible cursor at all, you're going by touch.
You need to know where you're touching once you enable "Zip Mode".

The interesting part which should be brought up is in the "all pages" category.
Due to an oversight in the code, we're able to jump around the Ages much like the PlayStation version... after you exit a different Age.
Selenitic is the first age you should be entering which will open the entire game.

At the cost of doing the worst puzzle.


PlayStation Portable

Now the PlayStation Portable version may be a nothing burger in terms of changes,
it does have an interesting choice made by the Myst community that needs to be brought up.

The community allows the back ups of the UMD releases to get into the game sooner thanks for the use of the Japanese version.
As well as allowing the use of the Vita with Adrenaline. Making it the best way to play for this handheld.


The Flash Release

So if you owned a PocketPC you had a chance to get Myst on it, and Riven.
There's unfortunately no documented information about this version in the speedrunning scene.


ScummVM (but we're still the Mohawk Engine)

Now we can just refer to the original comments about the Windows release, but it's real important to mention here.
The ScummVM version of Myst allows you to "skip" sequences thanks to the Escape key, or whatever you rebind it to.

In Myst there's "PC" and "SVM-Free".
To keep it short, PC is the official current release on Good old Games (GoG) and Steam.
Available for Windows and Mac. This is a special ScummVM build of Myst: Masterpiece Edition which is to be used.

"SVM-Free" is whatever version of ScummVM that you want to use,
from the aforementioned GoG and Stream release or your own Physical CD or DVD release of the title.
This category is also for the Linux runners out there.


Myst on the Nintendo 3DS

This is a bit of a different version, this is realMyst released for the Nintendo 3DS.

It follows the newer graphical artstyle that came out for its time,
but the movement is still that of point-and-click, no free exploration unlike actual versions of realMyst.

This is also the worst version of Myst. This isn't up for a debate.
You do not have touch support like the DS release of the title, instead you have to use the top screen,
whilst fighting the 3DS' analog stick due to the extreme sensitivity of the cursor.

All zips are retained in this version.
